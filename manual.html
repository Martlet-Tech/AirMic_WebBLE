User Manual v1.0

The flight recorder
for your drone's voice

AirMic is a compact audio recording module for FPV drones. It captures high-quality audio synchronized with your flight controller, with built-in Remote ID (RID) broadcasting, wireless file transfer, and OTA firmware updates.

What's in the Box
📦

Each AirMic module includes:

  • AirMic board — pre-flashed with the latest firmware
  • USB-C cable — for power, data, and MSC mode
  • microSD card (optional) — for recording storage
  • Mounting hardware — screws and vibration dampeners
💡 Note: A microSD card (Class 10 / UHS-I recommended) is required for recording. The card can be inserted before or after power-up.

3
Quick Start
Get recording in under 2 minutes
⚡
quickstart-flow.jpg
Visual flow: insert SD → power → press button → recording
Insert a microSD card

Format FAT32, Class 10 or faster. Push until it clicks.

Power the module

Connect to a 5V BEC or USB-C. The mono LED starts slow blinking — AirMic is ready.

Press the button once

The LED switches to fast blink and the RGB LED turns red. Recording has started.

Press again to stop

The file is saved to /sdcard/AM_xxxx.wav (or AAC, depending on settings).

✈️ Pro tip: When connected to a flight controller, AirMic automatically starts recording when the FC arms and stops when it disarms. No button needed.
4
BLE Connection

AirMic advertises as Martlet AirMic over BLE. Use the WebBLE control page to connect:

Open the WebBLE page

Navigate to the AirMic control page on any device with Chrome (Android/Desktop).

qr-code.png
Scan to open:
airmic.martlet-tech.com
Click to connect

Tap the top bar (Bluetooth icon). Choose Martlet AirMic from the device list.

Configure settings

Once connected, you can change encoder format (WAV / AAC / ALAC), channels, AGC mode, and set up WiFi.

📱 Supported browsers: Chrome (Android 8+ / Desktop), Edge (Android). iOS Safari does not support WebBLE. Use a laptop or Android device for BLE configuration.
5
WiFi Setup

WiFi is required for file transfers and OTA firmware updates. Configure it through the BLE connection:

⚙️
web-settings-tab.jpg
Screenshot of Settings tab showing WiFi fields, encoder select, and AGC mode
Connect via BLE

Follow section 4 to establish a BLE connection.

Enter your WiFi credentials

In the Settings tab, enter your 2.4GHz WiFi SSID and password. Click Connect WiFi.

Wait for connection

The WiFi icon turns cyan when connected. The IP address appears in the top bar. The page automatically switches to the Files tab.

⚠️ 5GHz not supported: AirMic uses ESP32-S3 with 2.4GHz WiFi only. Make sure your network is not hiding the SSID.
6
Recording Modes

Manual (Button) Recording

  • Press the button once to start recording — LED fast blinks
  • Press again to stop — LED returns to slow blink
  • Files named AM_xxxx_YYYYMMDD_HHMMSS.wav (or .aac)

FC Auto-Recording

🔌
fc-wiring.jpg
Diagram: AirMic UART TX/RX → Flight Controller UART RX/TX, power from BEC
  • Connect AirMic to your flight controller's UART (TX/RX)
  • When the FC arms, AirMic automatically starts recording with AGC in flight mode (0dB gain) and RID broadcasting
  • When the FC disarms, recording stops and the file is saved
  • If you press the button while FC is armed, the button press is ignored — FC mode has priority
🔊 AGC behavior: In flight (armed), AGC applies 0dB gain to capture prop noise. On the ground (disarmed), AGC applies +20dB gain to pick up voices at a distance. This setting is automatic — no user action required.

Encoder Formats

Configured via BLE → Settings tab:

WAVUncompressed, largest files, best compatibility
AACEfficient compression, good quality, smaller files (recommended)
ALACApple Lossless, compressed but perfect quality

File Management
📂
web-files-tab.jpg
Screenshot of the Files tab showing file list, storage bar, and action buttons

Access recordings through the WebBLE control page's Files tab (WiFi required):

  • Browse — all files listed with size and date
  • Play — stream audio directly in the browser
  • Download — save files to your phone or computer
  • Rename — organize your recordings
  • Delete — remove unwanted files (with confirmation)

USB Mass Storage (MSC) Mode

Plug AirMic into a computer via USB-C while the module is powered. It enters MSC mode, appearing as a standard USB drive. You can copy files directly with any file manager.

💡 MSC mode reboots the device. The module will restart after USB ejection. Make sure recording is stopped before connecting USB.
8
Remote ID (RID)

AirMic includes a Remote ID broadcast feature that transmits drone identification and position data.

⚠️ Irreversible operation: RID unlock uses eFuse OTP (one-time programmable) memory. Once unlocked, it cannot be reversed. Only proceed if required by your local regulations.
Connect via BLE

Go to the Settings tab and check the RID status.

Type UNLOCK

Type "UNLOCK" in the confirmation field to enable the unlock button.

Confirm in overlay

A confirmation dialog appears. Read the warning carefully. Once confirmed, the eFuse is burned and RID is permanently enabled.

After unlocking, RID broadcasting starts automatically whenever the flight controller is armed.

9
Firmware Update (OTA)

AirMic supports over-the-air firmware updates via WiFi. The WebBLE page automatically checks for new versions on the OTA tab.

Connect WiFi

OTA requires an active WiFi connection. Set up WiFi via BLE first (section 5).

Select a firmware file

Download the latest .bin from the GitHub Releases page, or use the update banner link if available.

Upload & Flash

Click Upload & Flash. The file is uploaded via HTTP POST. The device reboots automatically after flashing.

⚠️ Do not power off during OTA. Interrupting a firmware update can brick the device. Ensure the battery is charged or use a stable power source.

Troubleshooting
No LED activityCheck power (3.3-5V BEC or USB). The module requires 200mA peak.
Cannot find "Martlet AirMic" in BLE scanThe device only advertises when idle (not recording, not in MSC mode). Try rebooting.
WiFi connects but no IPCheck DHCP is enabled on your router. Try a different 2.4GHz network.
Recording starts but no file savedCheck the SD card is inserted and formatted FAT32. Try reformatting.
SD card not detectedTry a different card. AirMic supports SDHC/SDXC up to 256GB. Format FAT32 with 64KB allocation unit size.
File playback has no audioEnsure the microphone hole is not blocked. Check encoder setting — WAV is best for testing.
OTA upload failsCheck network stability. Try a wired connection on the AP side. ESP32-S3 supports 2MB firmware images.
Device stuck / unresponsivePress and hold the button for 10 seconds to force a reset, or disconnect and reconnect power.
